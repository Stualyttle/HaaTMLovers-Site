WHAT WE DO

Our goal is to make programming more accessible for people that cannot do it very well. Even for people that do not like to program alone we offer a solution. We have a discord server where people can code together, we offer the option to assist by answering questions to new programmers so they can continue working on their project, we do this because we find the existing forum system to be rather slow, All and all what we're trying to enable more people to code aswell as making it enjoyable.
